免費下載各作業系統上的行列輸入法，以及字根表圖片、字符表與編碼表、字型

下載行列

Windows 作業系統

Windows 作業系統有內建行列輸入法，但若電腦可以安裝軟體，建議下載新行列輸入法或是 DIME 行列來使用。

Windows 內建的行列輸入法

直接到控制台>>語言>>中文>>選項>>新增鍵盤，加上行列就可以了。

內建的行列目前尚未完整支援中日韓統一表意文字擴充區 B，也就是遇到罕用字可能無法輸入。除此之外，目前內建行列的介面設計與使用體驗不甚理想，也沒有輸入法設定可以調整功能。

簡單來說，系統內建的行列輸入法還算堪用，但除非您使用的電腦無法安裝軟體（例如公司電腦、公用電腦等），否則推薦使用新行列輸入法或是DIME 行列：支援更多字元的輸入，更佳的使用體驗，更豐富的輸入法設定。

新行列輸入法

新行列輸入法是 Windows 系統上推薦使用的行列輸入法。

下載

到這裡下載（或到作者於 PTT 的發文，內含相同的下載連結）。安全無毒，請放心下載與安裝。

安裝與啟用

下載並解壓縮後，請先閱讀 readme.txt。readme.txt 這份檔案裡面除了會說明安裝方式，還說明了新行列輸入法的功能（相當值得一看）。看完 readme.txt 後，點擊 Install.exe 就可以安裝新行列輸入法。

一樣到控制台>>語言>>中文>>選項>>新增鍵盤，加上新行列輸入法來啟用。

功能與設定

使用新行列輸入法時，在「中/英」圖示上按滑鼠右鍵即可進入輸入法設定（如果沒有成功，可能是您的 Windows 比較舊，請參考 readme.txt 以進入輸入法設定）。

新行列輸入法的功能在 readme.txt 內都有寫，不過這裡列出其中一些功能。

✔️ 更廣的擴充區字元輸入：進入輸入法設定，可以選擇啟用更廣的擴充區字元輸入，也就是可以輸入更多罕用字。一般使用預設狀態（支援到擴充區 A）、或是額外勾選擴充區 B 就足以應付多數狀況。

✔️ 聯想字詞：默認關閉，要進入輸入法設定勾選「提示相關字詞」。直接以數字鍵選取聯想字詞。

✔️ 中英切換：除了按住 Ctrl 不放同時按下 space 之外，還可以使用 Shift 鍵（可到設定調整）。

✔️ 特別碼提示：默認關閉，要進入輸入法設定勾選「提示特別碼」。

✔️ 詞彙輸入：內建詞彙輸入功能，不但已內建 17 萬 8400 條詞彙，還能自行新增詞彙。詞彙輸入功能預設是關閉，必須進入新行列輸入法的設定>>進階，勾選「啟用詞彙輸入」。

不認識行列輸入法的詞彙輸入？在本站的行列教學有詳細的說明！

新行列輸入法還有以下功能：

⭐️ emoji 選單：連按兩次符號鍵 2↑（也就是連按兩次 w 鍵），再按任意一顆數字鍵便會進入 emoji 選單（不同的數字鍵對應到不同類型的 emoji 選單，不過目前僅數字鍵 1 到 8 這八顆數字鍵有此功能）。

⭐️ 日文假名模式：按住 Alt 不放同時按下 ` 可以進入/退出日文假名模式（按鍵 ` 在數字鍵 1 旁左邊）。日文假名模式下，鍵入假名的羅馬拼音即可輸入假名，鍵入 kao 進入顏文字選單。

DIME 行列

DIME 行列是 Windows 系統上推薦使用的行列輸入法。

下載

點此直接下載，或到這裡下載。

安裝與啟用

解壓縮後直接點擊 .exe 檔安裝。安裝後如果出現其它不會使用到的輸入法（例如 DIME 自訂、DIME 大易等），可以到控制台>>語言>>中文>>選項>>新增鍵盤，加入所有 DIME 的輸入法，再移除不需要的輸入法

功能與設定

使用 DIME 行列時，按住 Ctrl 不放同時按下 \ 可以進入DIME 行列輸入法設定。

介紹幾個實用的功能：

✔️ 更廣的擴充區字元輸入：進入輸入法設定>>行列 Unicode 查詢範圍，可以選擇啟用更廣的擴充區字元輸入，也就是可以輸入更多罕用字。一般使用預設狀態（Unicode Extension-A，即支援到擴充區 A）、或是選 Unicode Extension-AB 就足以應付多數狀況。

✔️ 中英切換：除了按住 Ctrl 不放同時按下 space 之外，還可以使用 Shift 鍵（需到設定啟用）。

✔️ 輸出字元：可以選擇繁體中文或簡體中文。

✔️ 詞彙輸入：DIME 行列有詞彙輸入功能但不是正統詞彙輸入功能。詞彙以如同字的方式輸入，也就是說最後要按空白鍵，而不是 Enter 旁邊的' 鍵。缺點就是可能導致重碼數量變多，並且可能改變重碼排序。DIME 行列本身無內建詞彙，要到設定的「自建詞庫」自行新增詞彙編碼。可以到本頁面的下載字符表與編碼表下載新行列輸入法中的詞彙檔，直接使用 DIME 行列設定中的「匯入自訂詞庫」新增詞庫。

不認識行列輸入法的詞彙輸入？在本站的行列教學有詳細的說明！

✔️ 行列反查：使用 DIME 其它輸入法（自建、注音、大易）時，可以顯示行列編碼。需進入設定啟用。

RIME 輸入法（小狼毫 Weasel）、PIME 輸入法、gcin for Windows。

Linux 作業系統

Linux 系統預設使用 iBus 做為系統輸入法框架，但因 iBus 的行列輸入法引擎（input engine，即 ibus-table-chinese-array）有缺陷，所以建議不使用該引擎而改用 ibus-array 輸入法引擎。

安裝 ibus-array 的方式，依照您採用的 Linux Distribution 而有所不同：

Debian/Ubuntu

Debian/Ubuntu 內建 ibus-array：在終端機輸入 sudo apt update 後，輸入 sudo apt install ibus-array 即可完成安裝。可以透過在終端機輸入 ibus-setup 進入 ibus 偏好設定。

Fedora/RHEL/CentOS

Fedora/RHEL/CentOS 雖然沒有內建 ibus-array，但 bluebat 已將 ibus-array 打包好並置於第三方 RPM Sphere 的安裝套件庫中，請參照其說明，先載入 RPM Fusion 套件庫，然後安裝 rpmsphere-release 套件；此套件安裝好後，在終端機輸入 sudo dnf install ibus-array 即可完成 ibus-array 的安裝。

重新開機後，請在「設定值」的「地區和語言」中點選輸入來源的「+」按鈕，就可在「中文（臺灣）」中找到「中文(Array)」，最後點選「加入」，以將 ibus-array 加到您的輸入法清單中。

另外，Ubuntu 使用者除了 iBus 也可使用 gcin for Linux（已內建行列輸入法模組的輸入法框架）。由於系統架構的緣故，目前 gcin for Linux 無法安裝於 Fedora。

Mac OS 作業系統

Mac OS 作業系統上沒有內建行列輸入法，推薦下載香草輸入法：到這裡下載。

Android

Android 系統上沒有內建行列輸入法，推薦下載萊姆輸入法。不僅有行列 30 也有行列 10，並且支援繁簡轉換、字根反查等功能。

到 Play 商店搜尋「萊姆中文輸入法」，或是點此直接進入下載頁面。

在萊姆輸入法頁面上方點擊齒輪圖示可以進入設定頁面。設定頁面中，可以開啟反查功能。例如到注音字根反查選擇行列 10，在使用萊姆注音輸入時，系統就會提示輸入字的行列 10 編碼。另外，使用萊姆輸入法時，左右滑動鍵盤中的空白鍵可以快速切換不同的輸入法。

本站的行列教學有完整行列 10 的教學以及功能介紹呦！

另外還有 gcin for Android 可以使用。

iOS

iOS 系統上沒有內建行列輸入法，推薦下載 OkidoKey。不僅有行列 30 也有行列 10，並且支援繁簡轉換、字根反查等功能。

到 App Store 搜尋「OkidoKey」，或是點此直接進入下載頁面。

另外 iOS 系統上還有付費但功能更完整的 Frankie，能在不同輸入法之間快速切換（OkidoKey 一次只能載入一種輸入法）並且沒有廣告（OkidoKey 功能設定頁面可能會出現廣告）。到 App Store 搜尋「Frankie」，或是點此直接進入下載頁面。

現在付費的 Frankie 原名為 OkidoKey，而現在免費的 OkidoKey 原名為 OkidoKey Lite。

下載字型

本站使用網頁版思源宋體 Noto Serif CJK TC，僅支援到中日韓統一表意文字擴充區 A 區、以及 B 區中部分常用字。

如果要正確顯示多數罕用字以及網頁上的粗體，您可以免費下載此字體的完整版（點此前往下載頁）。

如果要正確顯示所有罕用字，您可以下載免費字體「花園明朝體 HanaMinB」（點此前往下載頁），此字體能夠完整支援到擴充區 F，不過一般來說不會有這個需求，下載完整版的思源宋體就足夠了。
